# Monkey Tag IO Guide

This repository is a small public resource note for [MonkeyTagIO.org](https://monkeytagio.org/), an unofficial browser play and guide hub for Monkey Tag IO.

Monkey Tag IO is a browser io tag game focused on chasing, climbing, jumping, and escaping tag pressure. The site keeps the player page simple, adds quick control notes, and collects related browser mini games for players who want another quick round.

Useful pages:

- [Play Monkey Tag IO](https://monkeytagio.org/)
- [How to play Monkey Tag IO](https://monkeytagio.org/how-to-play-monkey-tag-io/)
- [Monkey Tag IO controls](https://monkeytagio.org/monkey-tag-io-controls/)
- [Monkey Tag IO mobile notes](https://monkeytagio.org/monkey-tag-io-mobile/)
- [Related browser games](https://monkeytagio.org/games/)

This is not an official game repository. It exists as a clean citation and resource page for the public guide site.
